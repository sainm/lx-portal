# 心理咨询门户网站 MVP 实施计划

## 阶段 1：需求确认与信息架构

目标：确认第一版只解决获客、信任建立和预约线索收集。

产出：

- 页面清单
- 后台模块清单
- 预约表单字段
- 服务分类
- 咨询师资料字段
- 基础内容结构

建议确认项：

- 是否展示咨询价格
- 是否支持线上和线下咨询
- 是否需要企业心理服务入口
- 是否有固定咨询师团队资料
- 预约后由人工联系还是自动确认
- 是否接入网站统计、在线客服、地图或广告转化追踪

## 阶段 2：视觉与交互设计

目标：设计一个可信、安静、专业、低压力的门户体验。

设计重点：

- 首页首屏清晰表达服务对象和预约入口
- 咨询师信息易扫描
- 预约表单不要给用户造成压力
- 隐私与保密说明要容易看到
- 移动端优先，确保手机端预约顺畅

建议页面优先级：

1. 首页
2. 预约页
3. 咨询师列表页
4. 咨询师详情页
5. 服务项目页
6. 咨询流程页
7. FAQ 页
8. 文章列表与详情页
9. 关于我们

## 阶段 3：技术方案确认

目标：在开发前确认技术栈、部署方式和数据边界，避免开发中返工。

本项目后端默认采用 Java 21、Gradle、Spring Boot，详细架构见 `doc/technical-architecture.md`。

需要确认：

- 前端框架与 UI 方案
- 后端框架与接口规范
- 数据库选型
- 文件上传与图片存储方案
- 后台登录与权限方案
- 部署环境：云服务器、容器、平台托管或现有服务器
- 域名、备案、SSL 证书方案
- 日志、备份和错误监控方案
- 网站统计与埋点方案

建议 MVP 技术原则：

- 优先选择团队熟悉、上线快、维护成本低的方案
- 预约数据属于敏感信息，数据库和后台访问需要做好权限控制
- 图片和文章内容应便于运营人员维护
- 技术方案不为第二阶段过度设计，但要保留扩展支付、排班和用户中心的空间

## 阶段 4：前台开发

目标：完成门户展示与预约提交。

开发内容：

- 首页
- 服务项目页
- 咨询师团队页
- 咨询师详情页
- 预约表单
- 预约成功页
- 咨询流程页
- FAQ 页
- 文章列表页
- 文章详情页
- 关于我们页
- 隐私政策页
- 咨询须知页

关键验收：

- 用户可以从任意主要页面进入预约
- 预约表单可以正常提交
- 咨询师、服务、文章内容可以正确展示
- 移动端页面不出现文字溢出或布局错乱
- SEO 标题和描述能够按页面配置

## 阶段 5：后台开发

目标：让运营人员可以维护内容和处理预约。

开发内容：

- 登录入口
- 预约管理
- 咨询师管理
- 服务项目管理
- 文章管理
- 基础配置

关键验收：

- 可以查看并处理预约线索
- 可以新增和编辑咨询师
- 可以维护咨询风格、擅长方向、服务人群、价格与时长
- 可以新增和编辑服务项目
- 可以发布文章
- 可以配置公司联系信息、营业时间、隐私政策和 SEO 信息

## 阶段 6：通知与运营闭环

目标：避免预约线索丢失。

建议实现：

- 预约提交后给用户展示成功页
- 预约提交后通知运营人员
- 后台可以标记联系状态
- 运营人员可以记录备注

推荐通知方式：

- MVP 优先推荐企业微信或飞书 webhook，接入快，适合内部运营跟进
- 如需要触达用户，再考虑短信或邮件
- 短信涉及成本、模板审核和用户触达合规，可放在后续增强

## 阶段 7：内容录入、基础设施与上线准备

目标：用真实内容和可用环境完成上线前准备。

需要准备：

- 公司介绍
- 首页文案
- 服务项目文案
- 咨询师资料
- FAQ 内容
- 隐私政策
- 保密说明
- 咨询须知
- 首批科普文章

基础设施：

- 域名解析
- ICP 备案，如部署在中国大陆服务器
- SSL 证书和 HTTPS
- 生产数据库初始化
- 后台初始账号创建
- 定时备份策略
- 日志与错误监控

上线前检查：

- 所有预约入口可用
- 联系电话、微信二维码、地址、营业时间正确
- 咨询师资料无占位内容
- 隐私政策和咨询须知可访问
- 移动端适配正常
- 后台账号权限可用
- HTTPS 配置正常

## 阶段 8：上线后观察

目标：用真实数据决定第二阶段做什么。

重点指标：

- 首页访问量
- 咨询师详情页访问量
- 预约表单打开量
- 预约表单提交量
- 预约转化率
- 用户最常选择的咨询方向
- 用户最常选择的咨询方式
- 运营人员跟进时效

数据采集方案：

- 使用网站统计工具记录页面访问、来源渠道和基础转化
- 对关键动作埋点：点击预约、打开预约表单、提交预约、查看咨询师详情
- 后台按状态统计预约线索：待联系、已联系、已预约、已完成、已取消
- 预约转化率建议按“预约提交量 / 预约表单打开量”和“预约提交量 / 首页访问量”分别观察
- 如接入第三方统计工具，需要同步更新隐私政策和 Cookie 或统计授权提示

根据数据决定是否进入第二阶段：

- 在线支付
- 排班系统
- 用户中心
- 心理测评
- 智能咨询师匹配
- 企业 EAP 专区
